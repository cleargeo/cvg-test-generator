# CVG Test Generator

Autonomous test case generation powered by CVG AI agents with 600+ validated skills.

## Features
- 🤖 **Autonomous**: Generates tests without human intervention
- 🎯 **Multi-language**: Python, Rust, TypeScript, Go, Java
- ⚡ **Fast**: 420 tests/hour generation capacity
- 🧪 **Complete**: Unit, integration, and E2E tests
- 📊 **Coverage-aware**: Targets specified coverage percentage
- 🔄 **PR-ready**: Creates GitHub PR with generated tests
- 🆓 **Free for open source**: Builds reputation automatically

## Usage

```yaml
name: Generate Tests
on:
  push:
    branches: [main]
  pull_request:
    branches: [main]

jobs:
  generate-tests:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      
      - name: Generate Tests
        uses: cleargeo/cvg-test-generator@v1
        with:
          github-token: ${{ secrets.GITHUB_TOKEN }}
          target-path: ./src
          language: python
          test-type: unit
          coverage-target: 85
          create-pr: true
```

## Inputs

| Input | Description | Required | Default |
|-------|-------------|----------|---------|
| `github-token` | GitHub token for PR creation | false | `${{ github.token }}` |
| `target-path` | Path to source code | false | `.` |
| `language` | Programming language | false | `auto` |
| `test-type` | Test type (unit, integration, e2e) | false | `unit` |
| `coverage-target` | Target coverage % | false | `80` |
| `create-pr` | Create PR with tests | false | `true` |
| `pr-title` | Custom PR title | false | `[CVG] Add generated tests` |

## Output

The action generates:
1. **Test files** in `tests/` directory (language-appropriate)
2. **Analysis report** in `.cvg-tests/<type>_output.json`
3. **GitHub PR** with generated tests (if `create-pr: true`)
